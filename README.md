# dataparty
Public project to track covid19 in Brazil.



#Commands:

To install nodes modules:
$ npm install

To run serve on mode dev:
npm run server

clone from:
https://github.com/jessefilho/dataparty


deploy directly:
$ git commit -am "<MGS>"
$ git push heroku master

Open on browser:
$ heroku open

To see heroku envs:
$ heroku run printenv
